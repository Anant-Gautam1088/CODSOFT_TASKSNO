# 🤖 Rule-Based Chatbot | CodSoft Python Internship

A simple **Rule-Based Chatbot** developed using **Python** as part of the **CodSoft Python Programming Internship**. The chatbot interacts with users by recognizing predefined keywords and responding with appropriate messages using conditional statements.

---

## 📌 Project Overview

This chatbot demonstrates the fundamentals of conversational AI using a rule-based approach. It processes user input, matches it with predefined rules, and generates suitable responses. The project focuses on Python programming concepts such as loops, conditional statements, functions, and built-in modules.

---

## ✨ Features

- 👋 Greets users with friendly responses
- 💬 Answers common questions
- 🤖 Introduces itself
- 😂 Tells random jokes
- 🕒 Displays the current time
- 📅 Displays the current date
- 🙏 Responds to thank-you messages
- 🚪 Ends the conversation when the user types **bye**
- 🧠 Uses a clean function-based structure for better readability
